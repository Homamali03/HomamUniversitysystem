# University Management System

A web-based University Management System built with **ASP.NET Web API**, **HTML**, **CSS**, **JavaScript**, **jQuery**, and **SQL Server** following REST API architecture.

## Features

* Student management (Add and manage students).
* Department management.
* Dashboard for viewing and managing data.
* Role-based authentication and authorization.
* RESTful API architecture.
* Responsive and user-friendly interface.

## Technologies Used

* ASP.NET Web API
* HTML
* CSS
* JavaScript
* jQuery
* SQL Server
* REST API

## Project Overview

This system helps manage university operations by providing an interface for handling students and departments. It also includes a dashboard and different access permissions based on user roles.
